# MASTERPLAN PROMPT — UI/UX ONLY FOR MULTIMODAL GRAPH-RAG INTELLIGENCE APP

Build ONLY the **frontend UI/UX** for a modern enterprise intelligence platform.

DO NOT implement backend logic, APIs, databases, authentication, file processing, or graph database integration.

This is strictly a **UI shell** that AntiGravity will later connect to the real backend.

The UI must be fully modular, scalable, and easy for future backend integration.

---

# CORE PRODUCT WORKFLOW (VERY IMPORTANT)

Design the app around this exact user workflow:

```text
Upload Files
   ↓
Processing Pipeline
   ↓
Structured Data Collections
   ↓
Query Workspace
   ↓
Knowledge Graph Explorer
   ↓
Insight & Infographic Generator
   ↓
Export / Report
```

The UI must visually guide users through this flow.

This workflow is the most important design principle.

---

## 3) PROCESSING PIPELINE PAGE

VERY IMPORTANT PAGE.

Do NOT skip this.

This page visually shows AI processing stages.

Create a horizontal pipeline flow:

```text
Upload → Parse → Extract → Categorize → Graph Merge → Ready
```

Each uploaded file should have a progress pipeline card.

Example:

air_quality_report.pdf

* Uploaded ✓
* Text Parsed ✓
* Entity Extraction ✓
* Relationship Mapping ⏳
* Graph Merge ⏳

This page is critical for explainability.

Use animated progress indicators.

No backend logic.

